#  Space Escape Runner

**Space Escape Runner** is a simple yet engaging arcade-style mobile game where players navigate through space, avoid incoming obstacles, and collect power-ups to achieve the highest score possible. Built with **React Native** and **Expo**, the game features smooth controls and a fun, fast-paced gameplay experience.

##  Tech Stack

* React Native
* Expo
* JavaScript

##  Features

* Intuitive and responsive controls
* Endless space-themed gameplay
* Challenging obstacles to test your reflexes
* Collectible power-ups to boost your performance
* Clean and lightweight user interface

##  Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/space-escape-runner.git
   ```

2. Navigate to the project folder:

   ```bash
   cd space-escape-runner
   ```

3. Install the required dependencies:

   ```bash
   npm install
   ```

4. Start the Expo development server:

   ```bash
   npx expo start
   ```

5. Open the **Expo Go** app on your Android or iOS device and scan the QR code to play the game.
